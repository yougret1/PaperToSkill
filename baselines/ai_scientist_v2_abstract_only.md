# Abstract-Only Context: AI Scientist-v2

AI Scientist-v2 is an end-to-end agentic system for automated scientific
discovery. It formulates scientific hypotheses, designs and executes
experiments, analyzes and visualizes data, and writes scientific manuscripts.
Compared with AI Scientist-v1, it removes reliance on human-authored code
templates, generalizes across machine-learning domains, and uses progressive
agentic tree search managed by an experiment manager. It also adds a
vision-language model feedback loop for improving figures and captions. The
authors evaluated it by submitting three autonomous manuscripts to an ICLR
workshop, where one reached workshop acceptance-level scores.
